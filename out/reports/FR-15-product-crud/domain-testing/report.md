# FR-15 — Domain Testing Report

This file should contain the domain testing analysis for FR-15 Product Management CRUD, including:
- Identification of all input variables and their domains (name, price, category, description, auth token)
- Equivalence classes for each input (valid and invalid partitions), for each CRUD operation
- Test cases derived from each equivalence class, with test steps, expected results, actual results, and pass/fail status
- Coverage of access control: admin vs non-admin vs unauthenticated
